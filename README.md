# Professional Portfolio

A modern, responsive portfolio website built with React, TypeScript, and CSS.

## Features

- Responsive design that works on all devices
- Single Page Application (SPA) with smooth scrolling
- Modern UI with dark/light mode support
- Sections for About, Skills, Projects, Experience, and Contact
- Form validation and submission

## Sections

1. **Navigation** - Fixed header with links to all sections
2. **Hero** - Introduction with call-to-action buttons
3. **Skills** - Visual representation of technical skills
4. **Projects** - Showcase of featured projects with technologies used
5. **Experience** - Timeline of work experience and achievements
6. **Contact** - Contact form with social media links

## Development

### Prerequisites

- Node.js
- npm or yarn

### Installation

```bash
npm install
```

### Running the Development Server

```bash
npm run dev
```

### Building for Production

```bash
npm run build
```

## Customization

To customize this portfolio for your own use:

1. Update the content in each component file in `src/components/`
2. Replace placeholder images with your own
3. Update contact information in the Contact component
4. Modify colors in `src/index.css` to match your brand

## Technologies Used

- React 19
- TypeScript
- Vite
- CSS (no external libraries)

## Original Template Information

This project was initially created from the React + TypeScript + Vite template which provides a minimal setup to get React working in Vite with HMR and some ESLint rules.
